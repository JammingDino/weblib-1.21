ABOUT
web_lib is an HTML / CSS / JavaScript GUI framework for NeoForge mods. It lets you build fully interactive Minecraft GUI screens using a subset of the web stack – complete with a box-model layout engine, CSS cascade, a JavaScript bridge for Java callbacks, and multi-page navigation with browser-style history.

TECHNOLOGY
HTML Parser – Custom hand-written tokenizer and DOM builder supporting a broad subset of HTML5 elements.
CSS Engine – Full cascade with specificity, class / ID / pseudo-class selectors, and computed-style inheritance.
Layout Engine – Block, inline, inline-block, and flexbox layout with margin / padding / border box model.
Script Bridge – Lightweight expression evaluator that routes HTML event attributes to registered Java callbacks without a full JS runtime.
Multi-page Navigation – Browser-style back / forward history, ResourceLoader for external CSS and JS files, and configurable page loaders.
